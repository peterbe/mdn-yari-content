Tagging standard

  • All pages should have the {{Tag("API")}} and {{Tag("Reference")}} tag.
  • Pages linked to the Web Authentication API should have the tags {{Tag("Credential Management")}} and {{Tag("Credential Management API")}}.
  • For methods/properties belonging to an interface, the name of the interface should be added (e.g. {{Tag("Credential")}} for {{domxref("Credential.id")}}
  • Methods should have "{{Tag("Method")}}", properties need a "{{Tag("Property")}}" tag, interfaces should have the {{Tag("Interface")}} tag and constructors should have the {{Tag("Constructor")}} tag.
  • If pages need any type of work, a "Needs*" tag should be added. For instance, where there is no existing examples, the {{Tag("NeedsExample")}} tag should be added.
