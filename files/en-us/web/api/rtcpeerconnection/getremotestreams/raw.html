The RTCPeerConnection.getRemoteStreams() method returns an array of MediaStream associated with the remote end of the connection. The array may be empty.

Syntax

var mediaStreams[] = pc.getRemoteStreams();

The return value is an {{jsxref("Array")}} of {{domxref("MediaStream")}} objects.

Example

var streams = pc.getRemoteStreams();
for (var stream of streams) {
  console.log("Remote streams: " + stream.id);
}
