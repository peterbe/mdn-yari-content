Ferramentas de localização

Há diversas ferramentas úteis para você utilizar durantes seu trabalho com localização:

Verbatim
Utilizado para a tradução de strings em vários projetos Mozilla, incluindo a interface de usuário do MDN (bem como a interface de usuário do Firefox)
Transvision
Um utilitário fornecido pela Mozilla França, que permite pesquisar as ocorrências de uma cadeia de caracteres Inglês, encontrar todas as várias traduções para um local de destino que são utilizados em todo código Mozilla. Útil para encontrar as traduções preferidas para palavras ou frases.
