Description

O Valor de size é um inteiro representando a quantidade de elementos que o objeto Set possui. A set accessor function for size is undefined; you cannot change this property.

Examples

Using size

var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add('some text')

mySet.size; // 3
