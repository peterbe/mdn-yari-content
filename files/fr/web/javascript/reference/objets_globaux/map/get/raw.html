La méthode get() renvoie un élément d'un objet Map.

{{EmbedInteractiveExample("pages/js/map-prototype-get.html")}}

Le code source de cet exemple interactif est disponible dans un dépôt GitHub. Si vous souhaitez contribuez à ces exemples, n'hésitez pas à cloner https://github.com/mdn/interactive-examples et à envoyer une pull request !

Syntaxe

maMap.get(clé);

Paramètres

clé
Ce paramètre obligatoire est la clé de l'élément qu'on souhaite renvoyer.

Valeur de retour

Renvoie la valeur associée à la clé donnée ou {{jsxref("undefined")}} si la clé ne fait pas partie de l'objet Map.

Exemples

var maMap = new Map();
maMap.set("truc", "toto");

maMap.get("truc");  // Renvoie "toto".
maMap.get("machin");// Renvoie undefined.
