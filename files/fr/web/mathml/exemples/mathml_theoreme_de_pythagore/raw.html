Nous allons prouver le théorème de Pythagore: a 2 + b 2 = c 2

Nous pouvons le prouver de façon algebrique, en montrant que l'aire du grand carré est égale à l'aire du carré interieur, ajoutée à l'aire des 4 triangles : ( a + b ) 2 = c 2 + 4 ⋅ ( 1 2 a b ) a 2 + 2 a b + b 2 = c 2 + 2 a b a 2 + b 2 = c 2
