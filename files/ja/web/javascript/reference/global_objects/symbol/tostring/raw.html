特定のシンボルオブジェクトを表す文字列を返します。

構文

Symbol().toString();

説明

{{jsxref("Symbol")}}オブジェクトは{{jsxref("Object")}}オブジェクトのtoStringメソッドをオーバーライドします。{{jsxref("Object.prototype.toString()")}}を継承しません。Symbolオブジェクトに対して、toStringメソッドはオブジェクトの文字列表現を返します。

文字列の連結はありません

SymbolsでtoString()を呼ぶ間は、文字列の連結をすることはできません。:

Symbol("foo") + "bar";       // TypeError: Can't convert symbol to string

例

Symbol("desc").toString();   // "Symbol(desc)"

// well-known symbols
Symbol.iterator.toString();  // "Symbol(Symbol.iterator)

// global symbols
Symbol.for("foo").toString() // "Symbol(foo)"
