:read-write は CSS の疑似クラスで、ユーザーが編集できる要素 (input や textarea など) を表します。

/* 編集できる <input> 要素を選択 */
/* Firefox では接頭辞付きで対応 */
input:-moz-read-write {
  background-color: #bbf;
}

/* Blink/WebKit/Edge では接頭辞なしで対応 */
input:read-write {
  background-color: #bbf;
}

メモ: このセレクターは {{htmlElement("input")}} 要素だけを選択するものではありません。 {{htmlelement("p")}} 要素に {{htmlattrxref("contenteditable")}} が設定されているものなど、ユーザーが編集できるあらゆる要素を選択します。

{{csssyntax}}

例

HTML

<input type="text" value="ここに何でも好きなものを入力してください。">
<input type="text" value="これは読み取り専用の入力欄です。" readonly>
<p>これは普通の段落です。</p>
<p contenteditable="true">この段落は編集できます！</p>

CSS

input { min-width: 25em; }
input:-moz-read-write { background: cyan; }
input:read-write { background: cyan; }

p:-moz-read-write { background: lightgray; }
p:read-write { background: lightgray; }
p[contenteditable="true"] { color: blue; }
