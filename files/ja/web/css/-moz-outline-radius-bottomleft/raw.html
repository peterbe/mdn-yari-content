Mozilla アプリケーションにおいて、 CSS の -moz-outline-radius-bottomleft プロパティは、要素の輪郭線 ({{cssxref("outline")}}) の左下の角を丸くするために使用することができます。

{{cssinfo}}

例

HTML

<p>段落の左下の角を見てください。</p>

CSS

p {
  margin: 10px;
  border: solid cyan;
  outline: dotted green;
  -moz-outline-radius-bottomleft: 2em;
}
