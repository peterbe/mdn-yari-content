Mozilla アプリケーションにおいて、 CSS の -moz-outline-radius-bottomright プロパティは、要素の輪郭線 ({{cssxref("outline")}}) の右下の角を丸くするために使用することができます。

{{cssinfo}}

例

HTML

<p>段落の右下の角を見てください。</p>

CSS

p {
  margin: 5px;
  border: solid cyan;
  outline: dotted red;
  -moz-outline-radius-bottomright: 2em;
}
